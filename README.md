# 📦 CoreInventory

A smart warehouse management system built with Flask and MySQL.

## Features
- 📥 Receipts & 📤 Delivery management
- 🔄 Internal transfers & stock adjustments
- 📋 Move history tracking
- 📊 Dashboard with live stats
- 🤖 AI Inventory Assistant

## Tech Stack
- Python / Flask
- MySQL
- HTML / CSS / JavaScript

## Setup
1. Clone the repo
2. Install dependencies: `pip install flask mysql-connector-python`
3. Create a `.env` file with your DB credentials
4. Run: `python app.py`
